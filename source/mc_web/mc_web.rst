6. 虚拟机还原
-------------------

对单台终端，进行恢复。

* 第一步 : 登录云教室管理台，　点击：　资源－虚拟机　
* 第二步 : 选择所属教室，终端类型选择学生，点击搜索
* 第三步 : 找到，对应的终端编号。将虚拟机删除。
* 第四步 : 对应学生终端，点击图标，即可正常使用虚拟机。
